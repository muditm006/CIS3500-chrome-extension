Smart Skill Extractor

Instantly extract and organize the key skills from your resume and job postings.

View on GitHub Get the Extension

Features

  • Upload PDF resumes and extract top skills by category
  • Overlay matching skills on Handshake & LinkedIn job postings
  • Confidence scores based on frequency and position
  • React + TypeScript popup UI
  • No backend required—runs entirely in your browser

Install & Setup

  1. Clone the repo: git clone https://github.com/myronkharkover/CIS3500-chrome-extension.git
  2. Install dependencies: npm install
  3. Install OpenAI Package: npm i openai
  4. Build: npm run build
  5. Build Content Scripts: npm run build:content
  6. Load in Chrome: navigate to chrome://extensions
  7. Enable Developer mode & "Load unpacked" → select the dist folder
