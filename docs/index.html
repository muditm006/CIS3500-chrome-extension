About the Extension

Smart Skill Extractor is a Chrome extension designed to help job seekers better align their resumes with job postings on Handshake. By uploading a PDF resume, users can extract and categorize their top skills, which are then matched against the requirements listed in Handshake job posts.

The extension uses the ChatGPT API to provide contextual analysis, calculating a match score between your resume and the posting. Skills are overlaid directly on the job listing, providing a fast and intuitive way to assess your fit — all with zero backend or server dependencies.

Use Cases

  • Resume Review: Upload your resume to instantly see which skills stand out and how they’re categorized.
  • Job Matching on Handshake: While browsing jobs on Handshake, get a visual overlay of skills that match your resume.
  • Match Score Guidance: Use our ChatGPT-powered match score to evaluate how aligned a job is with your profile.
  • Skill Discovery: See domain-specific categories like Programming, Cloud, Web, and Tools for better self-assessment.
  • No Backend Required: All features run locally in your browser—keeping your data private and secure.

Key Features

  • Upload PDF resumes and extract skills using PDF.js
  • Automatic skill categorization by domain (e.g., Programming, Cloud, Web)
  • Overlay matching skills directly onto Handshake job postings
  • Real-time match score generation using ChatGPT
  • Modern UI built with React, TypeScript, and Vite
  • Client-side storage via Chrome Storage API for synced data across sessions
  • Smart error handling, prompt engineering, and performance optimizations

Install & Setup

  1. Clone the repo: git clone https://github.com/myronkharkover/CIS3500-chrome-extension.git
  2. Install dependencies: npm install
  3. Install OpenAI Package: npm i openai
  4. Build: npm run build
  5. Build Content Scripts: npm run build:content
  6. Load in Chrome: navigate to chrome://extensions
  7. Enable Developer mode & “Load unpacked” → select the dist folder
